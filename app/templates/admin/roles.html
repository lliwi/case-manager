Información sobre RBAC
Control de Acceso Basado en Roles (RBAC)

El sistema utiliza RBAC para gestionar permisos de forma centralizada. Los usuarios pueden tener múltiples roles asignados, y cada rol otorga un conjunto específico de permisos.

Roles del Sistema:
Admin
Acceso completo al sistema incluyendo gestión de usuarios, configuración y auditoría. Debe asignarse solo a personal autorizado.
Detective
Rol operacional para detectives privados. Permite gestionar casos, evidencias, análisis y generar informes periciales. Requerido para el trabajo diario.
Advertencia

Los cambios en roles y permisos tienen efecto inmediato. Asegúrese de que los usuarios tengan solo los permisos necesarios para sus funciones (principio de mínimo privilegio).
